About ICOS Search

ICOS Search is a privacy-focused search engine that provides Google search results without tracking, ads, or JavaScript requirements. Built with user privacy and data protection as core principles.

Key Features

  • No tracking or user data collection
  • Ad-free search results
  • Works without JavaScript
  • Customizable themes (Light/Dark/System)
  • Voice search support
  • AI-powered search insights
  • Multi-language support
  • Mobile-responsive design

Acknowledgments

Built on Whoogle Search

ICOS Search is built upon the excellent foundation of Whoogle Search, an open-source privacy-focused search engine. We extend our gratitude to the Whoogle community for their pioneering work in privacy-preserving search technology.

Open Source Libraries

This project wouldn't be possible without the incredible open-source community. Special thanks to:

  • Flask - Web framework for Python
  • BeautifulSoup4 - HTML parsing and processing
  • Font Awesome - Icons and visual elements
  • Gunicorn - Python WSGI HTTP Server
  • Cryptography - Security and encryption

Privacy Commitment

We believe privacy is a fundamental right. ICOS Search operates with zero logging, no user tracking, and complete data anonymization. Your searches remain private and secure.
