About ICOS Search

ICOS Search is a privacy-focused search engine that provides Google search results without tracking, ads, or JavaScript requirements. Built with user privacy and data protection as core principles.

Key Features

  • No tracking or user data collection
  • Ad-free search results
  • Works without JavaScript
  • Customizable themes (Light/Dark/System)
  • Voice search support
  • AI-powered search insights
  • Multi-language support
  • Mobile-responsive design
